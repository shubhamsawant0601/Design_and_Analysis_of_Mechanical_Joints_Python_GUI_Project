# Design and Analysis of Mechanical Joints - Python GUI Project
 Designing and Analysing the Mechanical Joints like Knuckle Joints and Cotter Joints Using Python GUI TKinter library

![image](https://user-images.githubusercontent.com/63664618/144641461-1669250f-a9d3-4d6b-bb02-78c5f59a6ffc.png)

## Second Window - According to the Joint selected, in next window fill the details regarding following...
### 1. Material of joint
### 2. Required Load Applications
### 3. Results will be displayed after submitting the details.
#### Dimensions of each parameters are founded which causes safe permissible amount of stress inside each component, also stresses generated in each components are calculated and shown in green if are under ultimate stress.
